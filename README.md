Основные функции:
- Регистрация и авторизация пользователей
- Добавление растений в свою коллекцию:
  Название растения
  Год посадки
  Фотография растения
  Инструкции по уходу
- Просмотр списка своих растений
- Удаление растений свайпом
- Сохранение состояния авторизации между запусками
